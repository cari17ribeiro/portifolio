FinTrack

Aplicação web para controle de finanças pessoais, construída com Python, Streamlit e SQLite. O projeto organiza receitas e despesas, importa extratos CSV/OFX com revisão segura e transforma movimentações em indicadores claros para apoiar decisões do dia a dia.

Apresentação do projeto

Objetivo

Centralizar o controle financeiro pessoal em uma interface simples, com visão de saldo atual, receitas, despesas, filtros por período e leitura segura de extratos.

Segurança

O processamento dos arquivos acontece localmente. O sistema não solicita senhas bancárias e evita duplicidades com assinatura SHA-256 dos lançamentos importados.

Qualidade

O projeto inclui testes automatizados das regras de negócio e fluxo de CI no GitHub Actions para validar o comportamento em Python 3.11 e 3.12.

Tecnologias

Python 3.11+ Streamlit Pandas SQLite Pytest GitHub Actions CSV OFX

Funcionalidades

Dashboard moderno com cards de saldo, receitas, despesas e total de lançamentos.
Cadastro manual de receitas e despesas por data, valor, descrição e categoria.
Importação de extratos CSV e OFX com etapa de revisão antes da gravação.
Detecção de lançamentos duplicados por assinatura SHA-256.
Configuração de saldo inicial e calibragem pelo saldo informado no extrato.
Filtros por período e tipo, tabela de lançamentos e gráfico por categoria.
